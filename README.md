# Proyecto-bases-de-datos
NeoPark ECCI es un sistema web desarrollado en Python/Flask con base de datos relacional normalizada en 3FN (7 tablas). Reemplaza el esquema manual del parqueadero de la Universidad ECCI Sede Centro automatizando el control de ingresos y salidas de vehículos, la asignación automática de espacios, el cálculo de cobros por tiempo de permanencia y la generación de reportes de ocupación y recaudo. El sistema implementa los temas del semestre: consultas avanzadas con vistas, seguridad por roles, funciones y procedimientos almacenados equivalentes a PL/SQL, triggers de validación y automatización, transacciones explícitas con COMMIT/ROLLBACK y control de concurrencia mediante threading.Lock y WAL journal mode.
